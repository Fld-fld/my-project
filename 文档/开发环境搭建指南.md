# 校园快递取件优化系统 - 开发环境搭建指南

## 1. 后端开发环境搭建

### 1.1 JDK 安装与配置

#### 1.1.1 下载 JDK

- **版本要求**：JDK 11.0.0.2（Oracle OpenJDK 11）
- **下载地址**：
  - Oracle 官网：https://www.oracle.com/java/technologies/javase/jdk11-archive-downloads.html
  - 或使用已安装的 JDK（确认版本即可）

#### 1.1.2 安装 JDK（Windows）

1. 双击下载的 JDK 安装包（如 `jdk-11.0.0.2_windows-x64_bin.exe`）
2. 选择安装路径（推荐：`D:\java\jdk-11.0.0.2`）
3. 等待安装完成

#### 1.1.3 配置环境变量

1. 右键"此电脑" → "属性" → "高级系统设置"
2. 点击"环境变量"
3. 在"系统变量"中新建：
   - 变量名：`JAVA_HOME`
   - 变量值：`D:\java\jdk-11.0.0.2`（你的 JDK 安装路径）
4. 编辑 `Path` 变量，添加：`%JAVA_HOME%\bin`

#### 1.1.4 验证安装

打开 PowerShell，执行：
```powershell
java -version
javac -version
```

输出应显示：
```
java version "11.0.0.2" 2024-07-02
OpenJDK Runtime Environment 18.9 (build 11.0.0.2+2-2)
OpenJDK 64-Bit Server VM 18.9 (build 11.0.0.2+2-2, mixed mode)
```

---

### 1.2 Maven 安装与配置

#### 1.2.1 下载 Maven

- **版本要求**：Maven 3.9.9
- **下载地址**：https://maven.apache.org/download.cgi
- 下载：`apache-maven-3.9.9-bin.zip`

#### 1.2.2 安装 Maven

1. 解压到安装目录（推荐：`D:\Maven\apache-maven-3.9.9`）
2. 无需运行安装程序

#### 1.2.3 配置环境变量

1. 新建系统变量：
   - 变量名：`MAVEN_HOME`
   - 变量值：`D:\Maven\apache-maven-3.9.9`
2. 编辑 `Path` 变量，添加：`%MAVEN_HOME%\bin`

#### 1.2.4 配置 Maven 镜像（加速下载）

编辑 `D:\Maven\apache-maven-3.9.9\conf\settings.xml`，在 `<mirrors>` 标签内添加：

```xml
<mirror>
  <id>aliyun</id>
  <name>Aliyun Maven</name>
  <url>https://maven.aliyun.com/repository/public</url>
  <mirrorOf>central</mirrorOf>
</mirror>
```

#### 1.2.5 验证安装

```powershell
mvn -version
```

输出应显示：
```
Apache Maven 3.9.9
Java version: 11.0.0.2, vendor: Oracle Corporation
```

---

### 1.3 IntelliJ IDEA 安装与配置

#### 1.3.1 下载 IDEA

- **版本**：IntelliJ IDEA 2024.x（Community 或 Ultimate 均可）
- **下载地址**：https://www.jetbrains.com/idea/download/

#### 1.3.2 安装 IDEA

1. 运行安装程序
2. 选择安装路径
3. 勾选"添加打开文件夹为项目"和"创建桌面快捷方式"

#### 1.3.3 配置 JDK

1. 打开 IDEA → `File` → `Project Structure` → `SDKs`
2. 点击 `+` → `Add JDK`
3. 选择 JDK 安装路径：`D:\java\jdk-11.0.0.2`

#### 1.3.4 配置 Maven

1. `File` → `Settings` → `Build, Execution, Deployment` → `Build Tools` → `Maven`
2. 设置：
   - Maven home directory: `D:\Maven\apache-maven-3.9.9`
   - User settings file: `D:\Maven\apache-maven-3.9.9\conf\settings.xml`

#### 1.3.5 安装必要插件

推荐安装：
- **Lombok**：简化 Java 代码
- **MyBatisX**：数据库开发辅助
- **GitToolBox**：Git 集成
- **Rainbow Brackets**：彩虹括号

## 2. 前端开发环境搭建

### 2.1 Node.js 安装与配置

#### 2.1.1 下载 Node.js

- **版本要求**：Node.js v22.15.0（或已安装版本）
- **下载地址**：https://nodejs.org/
- 下载 LTS 版本即可

#### 2.1.2 安装 Node.js

1. 运行安装程序
2. 选择安装路径（推荐：`C:\Program Files\nodejs`）
3. 勾选"Automatically install necessary tools"（可选）

#### 2.1.3 验证安装

```powershell
node -v
npm -v
```

输出应显示：
```
v22.15.0
11.6.0
```

#### 2.1.4 配置 npm 镜像（加速下载）

```powershell
npm config set registry https://registry.npmmirror.com
```

验证：
```powershell
npm config get registry
```

---

### 2.2 VS Code 安装与配置

#### 2.2.1 下载 VS Code

- **下载地址**：https://code.visualstudio.com/
- 下载 Windows 版本

#### 2.2.2 安装 VS Code

1. 运行安装程序
2. 选择安装路径
3. 勾选"添加到 PATH"和"创建桌面快捷方式"

#### 2.2.3 安装必要插件

打开 VS Code，点击左侧扩展图标，搜索并安装：

1. **Volar**（Vue 官方插件）
2. **Vue Language Features (Volar)**
3. **Prettier - Code formatter**（代码格式化）
4. **ESLint**（代码检查）
5. **Auto Rename Tag**（自动重命名标签）
6. **Path Intellisense**（路径智能提示）
7. **GitLens**（Git 增强）

#### 2.2.4 配置 Prettier

在 VS Code 中打开设置（`Ctrl + ,`），搜索 `format`，配置：

- `Default Formatter`: `esbenp.prettier-vscode`
- `Format On Save`: 勾选

## 3. 数据库环境搭建

### 3.1 MySQL 安装与配置

#### 3.1.1 下载 MySQL

- **版本要求**：MySQL 8.0.44
- **下载地址**：https://dev.mysql.com/downloads/mysql/
- 下载：`MySQL Installer for Windows`

#### 3.1.2 安装 MySQL

1. 运行 `mysql-installer-community-8.0.44.msi`
2. 选择 `Developer Default` 或 `Server only`
3. 设置 root 密码（建议：`123456`，开发环境简单即可）
4. 配置 MySQL 服务：
   - Service Name: `MySQL80`
   - 端口：`3306`
5. 完成安装

#### 3.1.3 验证安装

```powershell
mysql -u root -p
```

输入密码后，显示 `mysql>` 提示符表示成功。

---

### 3.2 创建项目数据库

```sql
-- 连接 MySQL
mysql -u root -p

-- 创建数据库
CREATE DATABASE IF NOT EXISTS campus_express 
CHARACTER SET utf8mb4 
COLLATE utf8mb4_unicode_ci;

-- 验证创建
SHOW DATABASES;
USE campus_express;
```

---

### 3.3 Navicat 安装（可选，推荐）

#### 3.3.1 下载 Navicat

- **版本**：Navicat Premium 16 或 Navicat for MySQL
- **下载地址**：https://www.navicat.com/

#### 3.3.2 连接 MySQL

1. 打开 Navicat
2. 点击"连接" → "MySQL"
3. 配置连接：
   - 连接名：`localhost`
   - 主机：`localhost`
   - 端口：`3306`
   - 用户名：`root`
   - 密码：`123456`（你的密码）
4. 点击"测试连接"，成功后保存

---

### 3.4 Redis 安装与配置（可选）

#### 3.4.1 下载 Redis

- **版本要求**：Redis 5.0.14.1（Windows 版本）
- **下载地址**：https://github.com/microsoftarchive/redis/releases
- 下载：`Redis-x64-5.0.14.1.msi`

#### 3.4.2 安装 Redis

1. 运行安装程序
2. 选择安装路径（推荐：`D:\Redis`）
3. 端口：`6379`

#### 3.4.3 启动 Redis

```powershell
# 启动 Redis 服务器
redis-server

# 启动 Redis 客户端
redis-cli

# 测试连接
redis-cli ping
# 应返回：PONG
```

## 4. 项目导入和运行

### 4.1 后端项目导入

#### 4.1.1 导入项目到 IDEA

1. 打开 IntelliJ IDEA
2. `File` → `Open`
3. 选择后端项目目录：`campus-express/backend`
4. 选择"Open as Project"

#### 4.1.2 等待 Maven 下载依赖

- IDEA 会自动识别 Maven 项目并开始下载依赖
- 首次下载可能需要较长时间（使用阿里云镜像会快很多）
- 等待右下角进度条完成

#### 4.1.3 配置数据库连接

编辑 `backend/src/main/resources/application.yml`：

```yaml
spring:
  datasource:
    url: jdbc:mysql://localhost:3306/campus_express?useUnicode=true&characterEncoding=utf8&useSSL=false&serverTimezone=Asia/Shanghai
    username: root
    password: root  # 修改为你的 MySQL 密码
    driver-class-name: com.mysql.cj.jdbc.Driver
```

#### 4.1.4 初始化数据库

1. 打开 `backend/src/main/resources/schema.sql`
2. 复制 SQL 脚本
3. 在 Navicat 或 MySQL 命令行中执行

#### 4.1.5 启动后端服务

**方法一：使用 IDEA**
1. 找到启动类：`src/main/java/com/example/express/ExpressApplication.java`
2. 右键 → `Run 'ExpressApplication'`

**方法二：使用 Maven 命令**
```powershell
cd campus-express/backend
mvn spring-boot:run
```

#### 4.1.6 验证启动成功

访问：http://localhost:8080/api/health

返回：
```json
{
  "code": 200,
  "message": "success",
  "data": "Server is running"
}
```

---

### 4.2 前端项目导入

#### 4.2.1 打开项目

1. 打开 VS Code
2. `File` → `Open Folder`
3. 选择前端项目目录：`campus-express/frontend`

#### 4.2.2 安装依赖

打开终端（`Ctrl + ``），执行：

```powershell
npm install
```

等待安装完成（首次安装可能需要几分钟）。

#### 4.2.3 配置 API 地址

创建或编辑 `frontend/.env` 文件：

```
VITE_API_BASE_URL=http://localhost:8080
```

> 注意：Vite 使用 `VITE_` 前缀，而不是 `VUE_APP_`

#### 4.2.4 启动前端开发服务器

```powershell
npm run dev
```

输出应显示：
```
  VITE v5.x.x  ready in xxx ms

  ➜  Local:   http://localhost:5173/
  ➜  Network: use --host to expose
```

#### 4.2.5 访问前端页面

浏览器打开：http://localhost:5173

## 6. 常见问题解决

### 6.1 后端启动失败

**问题 1：端口被占用**
```
Error: Port 8080 is already in use
```
解决：
```yaml
# 修改 application.yml
server:
  port: 8081  # 改为其他端口
```

**问题 2：数据库连接失败**
```
com.mysql.cj.jdbc.exceptions.CommunicationsException
```
解决：
- 检查 MySQL 服务是否启动
- 检查数据库用户名密码是否正确
- 检查数据库是否创建

**问题 3：Maven 依赖下载失败**
```
Could not resolve dependencies
```
解决：
- 检查网络
- 配置阿里云镜像
- 删除本地仓库重新下载：`C:\Users\你的用户名\.m2\repository`

---

### 6.2 前端启动失败

**问题 1：npm install 失败**
```
npm ERR! network timeout
```
解决：
```powershell
# 配置淘宝镜像
npm config set registry https://registry.npmmirror.com
# 删除 node_modules 重新安装
rm -r node_modules
npm install
```

**问题 2：端口被占用**
```
Error: Port 5173 is already in use
```
解决：
```powershell
# 修改 vite.config.js
server: {
  port: 5174  # 改为其他端口
}
```

**问题 3：跨域问题**
```
Access to XMLHttpRequest has been blocked by CORS policy
```
解决：
- 后端配置 CORS（项目已配置）
- 检查后端是否启动
