# 校园快递取件优化系统 - 系统详细设计文档

---

## 摘要

本校园快递取件优化系统使用 Java（Spring Boot）作为后端框架，前端采用 Vue.js + Element UI，数据库使用 MySQL，实现前后端分离的架构模式。该系统旨在解决校园快递站点取件拥堵、信息不透明、管理效率低等问题，为学生用户提供便捷的快递查询、取件预约服务，为站点管理员提供高效的快递管理后台。

**关键词：**校园快递；取件预约；前后端分离；Spring Boot；Vue.js

# 1 绪论

## 1.1 研究背景

随着电子商务的快速发展，校园快递业务量急剧增长，传统的快递取件模式已无法满足师生的需求：

- **取件拥堵**：高峰时段站点人流密集，学生排队时间长
- **信息滞后**：学生无法及时获知快递到达情况
- **管理低效**：手工录入和核对工作量大、易出错
- **缺乏预约**：无取件预约机制，导致拥堵加剧

因此，开发一套校园快递取件优化系统具有重要的现实意义。

## 1.2 研究意义

- **分流减压**：通过预约功能引导错峰取件，缓解站点压力
- **信息透明**：实时查询快递状态，减少无效等待
- **效率提升**：系统化管理快递信息，降低人工成本
- **体验优化**：消息通知、预约提醒等功能提升服务质量

## 1.3 研究现状

目前校园快递管理系统主要有以下类型：

1. **传统管理系统**：仅提供基础的快递录入和查询功能
2. **智能柜模式**：通过智能快递柜实现自助取件
3. **预约取件系统**：结合预约机制优化取件流程

本系统采用预约取件模式，结合智能通知功能，提供更优的用户体验。

## 1.4 研究内容

- 快递信息管理与取件码自动生成
- 取件预约与时间段管理
- 快递状态流转与消息通知
- 后台管理与数据统计

# 2 技术介绍

## 2.1 Spring Boot

Spring Boot 是一个用于构建独立、生产级别的 Spring 应用程序的框架。其主要特点包括：

- **自动配置**：根据依赖自动配置 Spring 应用
- **独立运行**：内嵌 Tomcat、Jetty 等服务器
- **简化依赖**：通过 Starter 简化依赖管理
- **社区支持**：丰富的社区资源和生态系统

## 2.2 Vue.js

Vue.js 是一个渐进式 JavaScript 框架，用于构建用户界面：

- **响应式数据绑定**：实现数据与视图的双向绑定
- **组件化开发**：支持组件复用和模块化
- **虚拟 DOM**：提高渲染性能
- **轻量级**：学习曲线平缓，易于上手

## 2.3 Element UI

Element UI 是基于 Vue.js 的组件库：

- **丰富组件**：提供表单、弹窗、表格等常用组件
- **美观设计**：现代化的 UI 设计风格
- **响应式布局**：支持不同屏幕尺寸
- **国际化支持**：多语言切换

## 2.4 MySQL

MySQL 是一款开源关系型数据库管理系统：

- **高性能**：优化的查询引擎
- **可靠性**：事务支持和数据完整性
- **扩展性**：支持主从复制和分区
- **开源免费**：降低开发成本

# 5 系统实现

## 5.1 用户功能模块的实现

### 5.1.1 登录页面

- **页面路径**：`/login`
- **功能描述**：用户输入手机号和密码进行登录
- **核心逻辑**：
  1. 表单验证（手机号格式、密码长度）
  2. 调用登录接口
  3. 存储JWT token到localStorage
  4. 跳转到首页

### 5.1.2 注册页面

- **页面路径**：`/register`
- **功能描述**：用户注册新账号
- **核心逻辑**：
  1. 表单验证（学号、姓名、手机号、密码）
  2. 检查手机号是否已注册
  3. 密码加密存储
  4. 注册成功后跳转到登录页

### 5.1.3 首页

- **页面路径**：`/`
- **功能描述**：首页展示和快递查询
- **核心逻辑**：
  1. 显示系统标题和查询框
  2. 输入取件码查询快递
  3. 展示快递详细信息

### 5.1.4 我的快递页面

- **页面路径**：`/myexpress`
- **功能描述**：查看个人快递列表
