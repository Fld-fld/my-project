## 6. 启动说明

### 6.1 后端启动

```bash
# 1. 导入数据库
mysql -u root -p123456 < backend/src/main/resources/schema.sql

# 2. 进入后端目录
cd backend

# 3. 启动
mvn spring-boot:run
```

### 6.2 前端启动

```bash
# 1. 进入前端目录
cd frontend

# 2. 安装依赖
npm install

# 3. 启动开发服务器
npm run dev

# 4. 访问
http://localhost:5173
```
